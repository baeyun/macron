Welcome to Macron Explorer v1.0.0!

This application contains the repository of all APIs supported by Macron 1.0.0. Not only are demos displayed, the full documentation of the Macron version is used. Each version of Macron comes with an explorer.

Getting Started!

In this section we'll try to explain how to get up and running as quick as possible with Macron.

Using the macron.config.js config file

The macron.config.js is a pretty basic file containing all the neccesities to acquire a fully 'macronized' window.

Implementing native extensions with C/C++ or Python

Creating a native extension itself is an easy task. Automating it is something else, and we'll get to that in the next section. As for creating simple extensions, we'll take a simple Python example.

Let's create a simple extension that just prints "Hello, World!" to an HTML page.

from macron import NativeBridge, MacronPrivateMethod

class HelloWorld(NativeBridge):
  def say_hello(self):
    return "Hello, World!"

exit() # Prevents this script from running after this point
